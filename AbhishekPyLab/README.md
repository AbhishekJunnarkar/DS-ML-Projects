# SETUP for MacOS

## Step 1: Python setup 

Verify Python3 - Comes installed on MacOS

$ python3 --version

## Step 2: Homebrew setup

$ brew --version

$ brew update

$ brew install python

## Step 3: Jupyter Notebook setup

$ brew install jupyterlab

$ jupyter --version

## Step 4: Scipy
Install
- brew install scipy

Verify command:
- python3 -c "import scipy; print(scipy.__version__)"
